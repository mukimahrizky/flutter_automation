## How To Run

### APPS

flutter build apk

flutter run

### TEST

flutter test test/widget_test.dart  

### TEST DRIVER

flutter drive --target=test_driver/e2e.dart

### TEST DRIVER BDD

flutter drive --target=test_driver/bdd.dart

### INTEGRATION TEST

flutter test integration_test/e2e_test.dart
